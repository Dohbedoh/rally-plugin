API Key with which plug-in will connect to Rally. If you do not have an API Key, you can do the following:

  1. Visit the Rally Login Page
  2. Log in using your Rally credentials
  3. Click on the API KEYS link in the menu bar at the top of the page
  4. Click on the Create New API Key link
  5. Give the API Key a name, select FULL ACCESS and click Create
  6. Copy the new API Key and paste
